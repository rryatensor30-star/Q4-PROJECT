### \#Project title

### Personal Habit and Deadline Tracker



### \##Project description

###### This program is a simple command-line planner that combines a habit tracker and a deadline manager, allowing users to log habits, track streaks, and manage tasks with due dates. It saves all data in a text file, updates habit streaks automatically, and displays task progress with statuses like pending, completed, or overdue.



### \##Installation

1.Start by finding your python file name.

2.Run the program/module and you'll be give 5 choices

3.Choices that are given is 1.Add Habit 2.Add deadline to task 3.Display 4.Complete/Remove Task 5. Exit

4.Each choice gives a different set of inputs from the other

&#x09;a. Add habit:

&#x09;- Lets you input a habit (e.g., “exercise”) and a date

&#x09;- If it’s a new habit → starts a streak at 1

&#x09;- If it already exists → increases the streak by +1

&#x09;b. Add deadline:

&#x09;- Lets you input:

&#x09;    Task name

&#x09;    Purpose/subject

&#x09;    Due date (YYYY-MM-DD)

&#x09;- Saves the task and marks it as not completed yet

&#x09;c. Display:

&#x09;- Shows all saved data in a table:

&#x09;	Habits → habit name and current streak

&#x09;	Deadlines → task, purpose, due date, days left, and status:

&#x09;	PENDING → not yet due

&#x09;	OVERDUE → past the due date

&#x09;	DONE → marked complete

&#x09;d. Complete/Remove Task:
	Completes/Removes the chosen task.
	e. Exit:

&#x09;- Exits the program

5.The data will save automatically into a txt file

6.You are allowed to input again and again until you want to leave the program.

7.When you're finally done with the program, type 5 to exit.

### 

### \##Usage of file:

It saves all your habits (with their streaks) and deadlines so that when you close and reopen the program, your data is not lost and can be loaded again.



### \##Features:

* Add and track daily habits with an automatic streak counter
* Create and manage tasks with deadlines with the ability to remove and add too
* Displays all habits and tasks in an organized table format
* Tracks task status (Pending, Overdue, Done) based on due dates
* Automatically saves and loads data using a text file
* Simple command-line interface for easy navigation





### \##Tech Stack/Built with:

* Python (3.13.2) – main programming language
* datetime module – handles dates and deadline calculations
* os module – checks and manages file storage
* Text File (.txt) – used for persistent data storage
